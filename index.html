Desenvolvimento de Jogos

Java

JavaScript, HTML, CSS, PHP

Com 2 anos de experiência em cibersegurança e projetos relativos ao curso nos quais fiquei, regularmente, destacado para desenvolver o back-end, mas também estive muito presente e contribui para o desenvolvimento do front-end also develop and contribute for the front-end com HTML+CSS e JavaScript com pedidos AJAX. C++

Eu tenho mais de 4 anos de experiência de programação em C++. Os meu projetos de C++ incluem o sistema de otimização de caminhos em grafos usando o algoritmo de Dijkstra e muitos outros projetos relacionados com encontrar os caminhos de menor peso e programas de otimização assim como jogos na CLI, como por exemplo o meu jogo path optimization system for graphs using Dijkstra's algorithm and many other projects relating to fastest path and optimization programs as well as some CLI games, such as my Maze Runner.

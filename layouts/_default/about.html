Systems engineer working across the stack — from ML pipelines in the cloud to neural networks running on bare-metal microcontrollers.

I'm a software engineer based in Portugal. My work spans production ML systems, GPU rendering pipelines, and embedded AI. I like the parts of the stack where the abstractions get thin — quantizing a model to fit in 128KB of flash, hand-managing Vulkan descriptor sets, or designing async event architectures that don't fall over under load.

Currently I build core infrastructure for Germany's largest digital tenant-protection platform. Before that I shipped production AI systems — RAG pipelines, LLM chatbots, and NLP classifiers deployed on-premises.
